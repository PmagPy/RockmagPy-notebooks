RockmagPy: tools for rock magnetic data analysis#

RockmagPy is a module of the PmagPy tools written in Python that is focused on the analysis of rock magnetic data. The functions within RockmagPy facilitate the visualization and interpretation of the results from experiments such as hysteresis loops, low-temperature remenance experiments, and thermomagnetic curves.

RockmagPy is designed to work with the MagIC database. It enables manipulation of MagIC data sets as well as preparation of contributions for uploading to MagIC. Example notebooks have been developed to aid in the use of RockmagPy and to enable well-documented data analysis. These notebooks can be downloaded from the RockmagPy notebooks repository to be run locally on oneâ€™s computer or run in the cloud on the EarthRef JupyterHub.

Jupyter notebooks using RockmagPy#

Notebooks have been developed to use RockmagPy to analyze data from rock magnetic experiments. The notebook versions that are shown within this book are static (i.e. they donâ€™t run the code). Some of the interactive features that have been developed are not visible as they would be in a running notebook.
